สร้าง function สำหรับ javascript เพื่อตรวจสอบว่า object นั้นๆ เป็น object ว่างหรือไม่
ในกรณีที่ object นั้นเป็น null หรือ undefined ผลลัพธ์จะเป็น false ในกรณีที่เป็น object และไม่เป็นค่าว่างคือ true
ในกรณีที่ส่งข้อมูลประเภทอื่นๆ มาที่ไม่ใช่ object เช่น number, string, boolean ให้แสดง throw Error
แสดงตัวอย่างการใช้งานแบบ ถูกต้องและแบบใส่ข้อมูลผิด
