Test case ที่ทดสอบมีดังนี้
1. ค่าเป็นเลขจำนวนเต็ม
2. เลขมีทศนิยมเป็นบวก
3. เลขมีทศนิยมเป็นค่าติดลบ
4. กรณี edge case สำหรับ 0
5. เมื่อส่งค่าเป็น None
6. กรณีเป็นเลขขนาดใหญ่
7. เป็นเลขทศยมที่ระหว่าง 0 - 1
8. กรณีเป็นเลขหลักร้อย ต้องไม่มี ,

def format_number(number: float) -> str:
    if number is None:
        return ""
    return f"{number:,.2f}"
