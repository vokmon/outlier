Request
จงสร้างรูปแบบ สำหรับใช้ใน csv เพื่อเก็บข้อมูลเกี่ยวกับราคาทองคำแท่งแบบ 96.5% รายวัน โดยที่ราคาเป็นราคาเปิดตลาดตอนเช้าเวลา 6.30 จากร้านค้าฮั่วเซงเฮงที่แสดงบน https://huasengheng.com
แสดงตัวอย่าง 5 แถวเพื่อให้รู้ว่าต้องใส่ข้อมูลอย่างไรบ้าง

csv นี้ใช้สำหรับเก็บข้อมูลราคาทองคำตามเวลาที่มีการเปลี่ยนแปลง โดยมีการบันทึก ราคา ณ เวลานั้น, วันที่ และ เวลาในรูปแบบ 24 ชั่วโมง ที่มีการเปลี่ยนแปลง ด้วยข้อมูลนี้ช่วยให้ผู้ใช้สามารถนำข้อมูลไปหาค่าเฉลี่ยของราคาทองคำในแต่ละวันตามที่ต้องการได้





Request
สร้างตารางเมนูอาหารที่แบ่งออกเป็น 3 มื้อหลัก ได้แก่ อาหารเช้า, อาหารกลางวัน และอาหารเย็น สำหรับวันจันทร์ถึงวันศุกร์ โดยเน้นเมนูอาหารไทยที่เหมาะสมสำหรับผู้ที่ต้องการลดน้ำหนัก

Response
ตารางเมนูอาหารไทยที่สามารถทำหรือหาซื้อได้ง่ายสำหรับ วันจันทร์ถึงศุกร์ สำหรับผู้ที่กำลังต้องการลดน้ำหนัก

```
| วัน     | อาหารเช้า              | อาหารกลางวัน               | อาหารเย็น                |
|--------|-----------------------|---------------------------|-------------------------|
| จันทร์   | ข้าวต้มปลา + ตำลึงลวก    | ต้มยำไก่ใส่เห็ด + ข้าวกล้อง     | แกงจืดเต้าหู้หมูสับ + ฟักทองนึ่ง |
| อังคาร  | ไข่ต้ม 2 ฟอง + กล้วยน้ำว้า | แกงเลียงกุ้งสด + ข้าวไรซ์เบอร์รี | น้ำพริกอ่อง + ผักลวก        |
| พุธ     | โจ๊กหมูสับใส่ไข่           | ยำวุ้นเส้นทะเล + ไข่ต้ม        | ต้มจับฉ่ายผักรวม            |
| พฤหัสบดี | ไข่กระทะ (ใช้น้ำทอด)     | แกงป่าปลาทูนึ่ง + ข้าวกล้อง     | ต้มจืดหัวไชเท้าใส่ไก่สับ       |
| ศุกร์    | กล้วยน้ำว้า + นมถั่วเหลือง  | แกงส้มดอกแค + ข้าวไรซ์เบอร์รี  | แกงจืดมะระยัดไส้หมูสับ       |

```

อาหารเหล่านี้ เหมาะสำหรับผู้ที่ต้องการลดน้ำหนัก เนื่องจากประกอบด้วยเมนูที่คำนึงถึงพลังงานและสารอาหารที่สมดุล ในแต่ละมื้อมี มีโปรตีนที่เพียงพอ มีใยอาหารสูงทำให้อิ่มนาน ลดอาหารประเภทใช้น้ำมันสูง และรสชาติอร่อย ช่วยให้การลดน้ำหนักเป็นไปได้อย่างมีประสิทธิภาพ
